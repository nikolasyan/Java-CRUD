Tecnologias usadas:

Java

O Java foi a tecnologia principal e mais usada. Responsavel por toda a funcionalidade e back-end do projeto.

JavaScript

O JavaScript foi utilizado somente para coisa simples. Como verificação de campos e uso de alertas nas páginas.

Bootstrap

O Bootstrap foi utilizado apenas para os elementos visuais do projeto. Dando um aspecto mais agradavel.

MySQL

O MySQL teve um dos papéis principais. Atuando como banco de dados ele foi responsável por armazenar e administrar todos os dados que são e serão utilizados pelo projeto.
